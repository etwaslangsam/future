
ある日、あなたは友達の女の子に呼び出されると‥‥


■ゴースト「みらいのふろあ」


■謝意

	このゴーストは ゆゆぴか さんの
	フリーシェル「コンフィズリー」
	を使用させて頂きました。この場で感謝。
	
▼概容

	未来から来た女の子とクラスメートの女の子が会話するゴースト
	フェーズ（イベント）進行・好感度進行などはありません
	あなた（ユーザー）は男性固定を想定して制作しています

	
▼キャラクター

	\0（右）：えぬ子
	自称・未来少女。本名不明。
	あなたとふろあと関わりがあるらしいが‥‥。
	
	\1（左）：神楽坂ふろあ
	あなたのクラスメートの女の子。
	ある日、放課後に彼女に呼び出されたことから物語が始まる。
